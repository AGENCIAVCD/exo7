# EXO7 Build

Website premium para a EXO7 Engenharia Integrada, construído com Next.js App Router, TypeScript, Tailwind CSS, Framer Motion e assets isométricos 3D da marca.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Verificação de produção

```bash
npm run typecheck
npm run build
```

## Estrutura

- `design.md`: design system completo da EXO7.
- `app/`: layout, SEO e página principal.
- `components/`: componentes reutilizáveis.
- `public/assets/`: imagens normalizadas para uso com `next/image`.
